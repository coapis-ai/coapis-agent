## 一、领域概述

### 1.1 领域定义

政府业务技能涵盖党政机关、政府部门、事业单位等政府机构的日常工作和专业业务，包括政策制定、公文写作、项目管理、行政审批、规划编制、监督检查等核心职能。

### 1.2 领域特征

| 特征 | 描述 |
|------|------|
| **规范性** | 遵循国家法规、部门规章、工作规范 |
| **严肃性** | 政治敏感、表述严谨、用语规范 |
| **专业性** | 需要专业领域知识、政策法规知识 |
| **层级性** | 不同层级政府有不同职责和权限 |
| **时效性** | 政策法规持续更新，需要及时跟进 |

### 1.3 用户群体

| 用户类型 | 典型部门 | 主要需求 |
|---------|---------|---------|
| **党政机关** | 党委、政府办公室 | 公文写作、会议管理、督查督办 |
| **职能部门** | 发改、财政、人社等 | 业务审批、项目管理、规划编制 |
| **基层单位** | 街道、社区 | 政策执行、便民服务、信息报送 |
| **事业单位** | 医院、学校、科研院所 | 业务管理、报告撰写、项目申报 |

### 3.1 技能：政策文件起草（gov_policy_draft_001）

#### 基础信息

```yaml

skill_level: "专业"
```

#### 技能描述

**简要描述**：根据政策需求和工作要求，起草符合规范的政策文件，包括办法、规定、实施意见等不同类型的政策文件。

**详细说明**：政策文件起草是政府工作中的核心任务之一，需要综合运用政策法规知识、公文写作规范、专业领域知识，生成符合要求的政策文件初稿。

**适用场景**：
- 新政策制定
- 政策修订完善
- 政策转化落地

**不适用场景**：
- 需要深度专业评估的政策
- 涉及重大利益调整的政策
- 需要多方协调的政策

| **必备知识** | 政治理论 | 熟悉 | 知识库 |
| **推荐知识** | 地方政策特点 | 了解 | 知识库 |

#### 能力要求

| **核心能力** | 逻辑组织能力 | 高级 | 合理组织政策条款和结构 |
| **核心能力** | 专业术语应用 | 高级 | 准确使用政策术语和专业表述 |
| **核心能力** | 政策依据引用 | 高级 | 准确引用上位法依据和政策基础 |
