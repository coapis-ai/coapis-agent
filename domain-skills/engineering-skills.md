## 一、领域概述

### 1.1 领域定义

工程业务技能涵盖建筑设计院、工程公司、施工企业、监理公司等工程机构的核心业务，包括工程设计、工程管理、工程造价、工程质量、安全管理等工程专业服务。

### 1.2 领域特征

| 特征 | 描述 |
|------|------|
| **专业性** | 需要工程专业知识和执业资质 |
| **安全性** | 涉及工程安全，错误容忍度极低 |
| **规范性** | 遵循工程规范、标准、规程 |
| **复杂性** | 涉及多专业协同、多环节配合 |
| **责任性** | 工程责任重大，终身责任制 |
| **技术性** | 需要专业工具和技术手段 |

### 1.3 用户群体

| 用户类型 | 典型机构 | 主要需求 |
|---------|---------|---------|
| **设计工程师** | 设计院、咨询公司 | 工程设计、方案编制、图纸绘制 |
| **项目经理** | 工程公司、施工企业 | 项目管理、进度控制、成本管理 |
| **造价工程师** | 造价咨询公司 | 工程造价、预算编制、结算审核 |
| **监理工程师** | 监理公司 | 质量控制、进度监督、安全监管 |

### 1.4 重要声明

⚠️ **工程业务技能使用声明**：

1. **辅助性质**：所有工程业务技能均为**辅助性质**，不能替代工程师的专业判断
2. **资质要求**：工程设计和审核需要**相应执业资质**
3. **责任明确**：工程质量责任实行**终身责任制**，责任由执业工程师和工程单位承担
4. **安全第一**：涉及工程安全的内容必须经过**专业人员审核确认**
5. **规范更新**：工程规范标准持续更新，使用时需确认规范有效性
