## 一、领域概述

### 1.1 领域定义

教育业务技能涵盖学校、培训机构、教育管理部门等教育机构的核心业务，包括教学设计、课程开发、学生评价、教学管理、教育研究等教育专业服务。

### 1.2 领域特征

| 特征 | 描述 |
|------|------|
| **专业性** | 需要教育学专业知识和教学技能 |
| **个性化** | 因材施教，关注个体差异 |
| **系统性** | 教学体系完整，循序渐进 |
| **创新性** | 教学方法持续创新 |
| **伦理性** | 遵守教育伦理，保护学生权益 |
| **发展性** | 关注学生全面发展 |

### 1.3 用户群体

| 用户类型 | 典型机构 | 主要需求 |
|---------|---------|---------|
| **教师** | 学校、培训机构 | 教学设计、课件制作、学生评价 |
| **教育管理者** | 学校、教育局 | 教学管理、质量评估、决策支持 |
| **课程开发者** | 教育公司、出版社 | 课程开发、教材编写、资源建设 |
| **研究人员** | 高校、研究机构 | 教育研究、政策分析、理论探索 |
