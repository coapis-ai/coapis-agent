## 一、领域概述

### 1.1 领域定义

金融业务技能涵盖银行、证券、保险、基金、信托等金融机构的核心业务，包括风险管理、信贷业务、投资分析、合规审查、金融分析、客户服务等专业职能。

### 1.2 领域特征

| 特征 | 描述 |
|------|------|
| **专业性** | 需要专业金融知识和资质 |
| **规范性** | 遵循金融法规和监管要求 |
| **风险性** | 涉及资金安全和风险管理 |
| **时效性** | 市场变化快，需要实时响应 |
| **合规性** | 监管要求严格，合规性要求高 |

### 1.3 用户群体

| 用户类型 | 典型机构 | 主要需求 |
|---------|---------|---------|
| **银行业** | 商业银行、政策性银行 | 信贷审批、风险管理、客户服务 |
| **证券业** | 证券公司、基金公司 | 投资分析、研究报告、合规审查 |
| **保险业** | 保险公司、保险经纪 | 产品设计、理赔审核、风险评估 |
| **其他金融** | 信托公司、租赁公司 | 业务审批、项目管理、风控合规 |

#### 技能描述

**简要描述**：对借款人或交易对手的信用状况进行评估，识别信用风险，提供风险评级和建议。

**详细说明**：信用风险评估是金融风险管理的核心环节，需要综合分析财务数据、征信记录、行业状况、经营情况等多维度信息，运用专业模型和经验判断，评估信用风险水平。

**不适用场景**：
- 需要实地调查的深度评估
- 涉及复杂担保结构的评估
