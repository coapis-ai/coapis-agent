## 一、领域概述

### 1.1 领域定义

医疗业务技能涵盖医院、诊所、卫生院等医疗机构的核心业务，包括临床诊断辅助、医疗管理、病历管理、医疗质量控制、药品管理等专业职能。

### 1.2 领域特征

| 特征 | 描述 |
|------|------|
| **专业性** | 需要医学专业知识和临床经验 |
| **安全性** | 涉及生命安全，错误容忍度极低 |
| **规范性** | 遵循医疗规范、诊疗指南、操作规程 |
| **伦理性** | 涉及医疗伦理、患者隐私保护 |
| **责任性** | 医疗责任重大，需要明确责任归属 |
| **时效性** | 急诊急救需要快速响应 |

### 1.3 用户群体

| 用户类型 | 典型机构 | 主要需求 |
|---------|---------|---------|
| **临床医生** | 医院、诊所 | 辅助诊断、病历书写、医学知识查询 |
| **护理人员** | 医院、养老院 | 护理评估、护理计划、护理记录 |
| **医技人员** | 医院、检验中心 | 检验报告、影像分析、病理诊断 |
| **管理人员** | 医院、卫生院 | 质量管理、制度制定、绩效评估 |

### 1.4 重要声明

⚠️ **医疗业务技能使用声明**：

1. **辅助性质**：所有医疗业务技能均为**辅助性质**，不能替代医生的诊断和治疗决策
2. **人工审核**：所有医疗相关输出必须经过**持证医疗专业人员审核**
3. **责任明确**：医疗行为的责任由**执业医师和相关医疗机构承担**
4. **患者知情**：使用AI辅助医疗服务时，应告知患者并获得同意
5. **监管合规**：严格遵守医疗卫生法律法规和医疗伦理规范
