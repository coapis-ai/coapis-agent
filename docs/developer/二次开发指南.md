## 开发环境搭建

### 前置要求
- Python 3.11+
- Node.js 18+ + npm
- Docker & Docker Compose
- Git

### 本地开发模式

```bash
# 1. 克隆项目
git clone https://github.com/coapis/coapis.git
cd coapis-agent

# 2. 配置环境变量
cp docker/.env.example docker/.env
# 编辑 docker/.env

# 3. 启动后端 Docker 服务
cd docker && docker compose up -d server

# 4. 启动前端开发服务器
cd ../client
npm install
npm run dev

# 5. 访问 http://localhost:5173
```

## 前端开发

### 技术栈
- React 18 + TypeScript
- Ant Design 5 (UI 组件库)
- Zustand (状态管理)
- Vite 5 (构建工具)

### 开发命令
```bash
cd client
npm install          # 安装依赖
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run lint         # 代码检查
npm run test         # 运行测试
```

### 添加新页面
1. 在 `src/pages/` 创建页面组件
2. 在 `src/App.tsx` 添加路由
3. 在 `src/i18n/` 添加翻译

## 后端开发

### 技术栈
- Python 3.11+
- FastAPI + Uvicorn
- SQLite/JSON 存储

### 开发命令
```bash
cd server
pip install -e ".[dev]"    # 安装开发依赖
uvicorn coapis.app:app --reload  # 启动开发服务器
python -m pytest tests/ -v       # 运行测试
black coapis/                     # 代码格式化
ruff check coapis/ --fix          # Lint 检查
```

### 添加新 API 路由
1. 在 `server/coapis/api/` 创建路由文件
2. 在 `app.py` 注册路由
3. 添加认证中间件

## 自定义技能开发

### 技能结构
```
skills/my-skill/
├── SKILL.md          # 技能说明（必须）
├── script.py         # 技能脚本
└── config.json       # 技能配置
```

### SKILL.md 格式
```markdown
# My Skill

## 描述
简要说明技能功能

## 触发条件
描述何时触发此技能

## 使用方法
如何使用此技能

## 参数
参数说明
```

### 技能注册
将技能目录放入 `skills/` 或用户工作区的 `skills/` 目录。

## 构建与部署

### 前端构建
```bash
cd client
npm run build
# 产物在 dist/ 目录
```

### Docker 构建
```bash
# 完整构建
docker compose -f docker/docker-compose.build.yml up -d --build

# 仅重建后端
docker compose build server
docker compose up -d server
```

### 生产部署
```bash
# 1. 构建前端
cd client && npm run build && cd ..

# 2. 构建 Docker 镜像
docker compose -f docker/docker-compose.build.yml build

# 3. 启动服务
docker compose -f docker/docker-compose.build.yml up -d
```

## 常见开发场景

### 添加新的内置工具
1. 在 `server/coapis/agent/tools/` 创建工具文件
2. 在 `tools/__init__.py` 注册
3. 更新工具描述和参数定义

### 修改 Agent 行为
1. 编辑 `server/coapis/agent/core.py`
2. 修改 Prompt 模板
3. 调整工具调用逻辑

### 添加新的用户等级
1. 修改 `server/coapis/api/auth.py` 中的等级逻辑
2. 更新环境变量配置
3. 前端调整等级显示
