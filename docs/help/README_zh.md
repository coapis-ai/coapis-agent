## 🌍 多语言支持

CoApis 支持以下语言：

| 语言 | 代码 | 状态 |
|------|------|------|
| 中文（简体） | `zh` | ✅ 完整支持 |
| English | `en` | ✅ 完整支持 |
| 日本語 | `ja` | ✅ 完整支持 |
| Русский | `ru` | ✅ 完整支持 |

在界面右上角点击 🌐 图标可切换语言。
