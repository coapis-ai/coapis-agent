# CoApis 产品概述

## 🦀 什么是 CoApis？

**CoApis** 是一款**多用户服务端 AI 智能体平台**，支持完全私有化部署。与面向个人用户的 AI 助手不同，CoApis 的核心定位是**在服务器端运行和管理多个 AI 智能体，供组织内多名用户共享使用**。

### 核心理念

```
传统 AI 助手：一个用户 → 一个 Agent → 个人使用
CoApis：    多个用户 → 多个 Agent → 组织共享
```

## 🌟 核心亮点

### 1. 多用户服务端智能体

- **Agent 运行在服务器端**：持续运行、后台执行定时任务、跨会话保持记忆
- **多用户共享**：组织内多名用户可以同时使用同一个 Agent
- **7×24 小时在线**：Agent 不依赖用户前端连接，始终在服务器端运行

### 2. 多租户数据隔离

- **独立工作空间**：每个用户拥有独立的 Agent、文件、记忆空间和偏好设置
- **权限控制**：精细的权限管理系统，支持角色和权限分配
- **数据隐私**：用户间数据完全隔离，确保数据安全

### 3. 越用越聪明的智能体

- **不会遗忘** — 三层记忆架构（长期记忆 / 领域知识 / 对话上下文），Agent 记住每一次对话中学到的东西
- **自动成长** — 从每次交互中提取经验，自动沉淀为可复用的知识和技能
- **技能进化** — 使用频率高、效果好的技能自动晋升到更高层级，低效技能自动淘汰
- **跨 Agent 传播** — 一个 Agent 验证有效的技能，可自动推广给其他 Agent

### 4. 技能生态

- **自动生成** — Agent 根据用户需求自动创建新技能，无需手动编写
- **持续优化** — 技能使用效果被追踪评估，触发词和内容随时间自动调优
- **版本管理** — 每次技能变更都有版本记录，可随时回滚到历史版本
- **三级技能体系** — 全局技能 → 用户技能 → 智能体技能，逐层覆盖，灵活定制

### 5. 企业级功能

- **审计日志**：完整记录所有操作
- **权限控制**：RBAC 权限管理系统
- **文件管理**：支持文件上传、下载、版本管理
- **定时任务**：支持 Cron 表达式定时任务
- **备份恢复**：支持数据备份和恢复

### 6. 多语言支持

- **中文**（简体中文）
- **English**（英文）
- **日本語**（日文）
- **Русский**（俄文）

## 📊 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript + Ant Design 5 + Vite 5 |
| 后端 | Python 3.11+ + FastAPI + SQLite + Pydantic v2 |
| 部署 | Docker + Docker Compose + Nginx |
| LLM | 任何 OpenAI 兼容 API (Ollama/OpenAI/Anthropic/vLLM 等) |

## 🎯 适用场景

- **企业内部知识库**：构建企业专属的 AI 知识助手
- **客户服务**：7×24 小时自动客服
- **研发辅助**：代码审查、文档生成、问题排查
- **数据分析**：自动数据分析、报告生成
- **运维管理**：智能监控、故障诊断、自动修复
