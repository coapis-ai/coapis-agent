# 办公场景二级分类设计方案

## 一、设计理念

### 1.1 核心思想

**按企业应用领域划分，而非功能场景**

```
❌ 错误思路（按功能场景）：
会议管理、公文处理、报告编制、审批流程...

✅ 正确思路（按应用领域）：
OA办公、审批服务、CRM客户管理、ERP资源管理、HR人力资源...
```

**为什么这样设计？**
1. **符合用户认知**：用户熟悉OA、审批、CRM、ERP这些概念
2. **便于分组显示**：一级分类作为分组，二级分类展示具体场景
3. **可扩展性强**：每个领域可以独立扩展，互不影响
4. **对齐商业产品**：钉钉、飞书、企业微信都是这样分类的

## 二、详细设计

### 2.1 OA办公 🏢

**定位**：日常办公协作，覆盖所有员工

**适用场景**：
- 政府：公文流转、会议组织、通知发布
- 企业：邮件沟通、日程安排、办公文档
- 学校：通知公告、会议纪要、工作汇报

**二级场景**：

| 场景ID | 场景名称 | 核心能力 | 典型应用 | 频率 |
|--------|---------|---------|---------|------|
| email-composing | 邮件撰写 | 邮件模板、语气调整 | 商务邮件、通知邮件 | 高 |
| schedule-management | 日程管理 | 日历管理、提醒设置 | 会议安排、任务提醒 | 高 |
| meeting-organization | 会议安排 | 会议通知、参会提醒 | 例会、专题会、培训会 | 高 |
| notice-announcement | 通知公告 | 模板库、格式规范 | 通知发布、公告起草 | 中 |
| meeting-minutes | 会议纪要 | 录音转写、纪要生成 | 会议记录、待办提取 | 高 |
| document-drafting | 公文起草 | 模板库、格式规范 | 通知、请示、报告 | 高 |
| work-report | 工作报告 | 结构化写作、数据整合 | 日报、周报、月报 | 高 |
| contact-management | 通讯录管理 | 联系人整理、分组管理 | 客户联系、同事通讯 | 中 |

**总计**：8个场景

  - 各领域的主要场景

- **低频场景**（9个）：偶尔使用，按需使用

## 五、实施建议

### 5.1 分阶段实施

**第一阶段**（基础场景，18个）：
- ✅ OA办公（6个）：邮件、日程、会议、通知、纪要、公文
- ✅ 审批服务（4个）：费用、请假、项目、合同
- ✅ 数据分析（3个）：数据统计、报表生成、KPI监控
- ✅ 知识管理（1个）：FAQ生成
- ✅ CRM（2个）：客户跟进、销售线索
- ✅ HR（1个）：考勤管理

**第二阶段**（扩展场景，20个）：
- OA办公（+2个）、审批服务（+4个）、CRM（+4个）
- ERP资源管理（6个）、项目管理（4个）

**第三阶段**（高级场景，14个）：
- HR人力资源（+5个）、知识管理（+5个）、数据分析（+3个）、项目管理（+2个）

### 5.2 与现有场景的关系

**现有场景**（4个办公场景）：
- meeting-minutes → OA办公 → 会议纪要
- document-drafting → OA办公 → 公文起草
- work-report → OA办公 → 工作报告（可合并到办公文档）
- project-approval → 审批服务 → 项目审批

**现有场景**（33个行业场景）：
- 保持独立，不纳入"办公"菜单
- 按行业领域分类（自然资源、生态环境、农业农村等）

## 六、优势分析

### 6.1 对比之前的方案

| 维度 | 之前方案（按功能场景） | 现在方案（按应用领域） |
|------|---------------------|---------------------|
| **分类逻辑** | 功能导向 | 应用导向 |
| **用户认知** | 需要理解具体功能 | 直接对应企业应用 |
| **分组效果** | 平铺展示 | 层次分明 |
| **可扩展性** | 功能间关联弱 | 领域内扩展强 |
| **商业对标** | 不符合主流产品 | 对齐钉钉、飞书 |

### 6.2 核心优势

1. **符合用户心智模型**
   - 用户熟悉OA、审批、CRM、ERP
   - 降低认知成本，快速找到场景

2. **便于分组显示**
   - 一级分类作为分组，二级分类展示场景
   - 层次清晰，不会"一屏显示不完"

3. **可独立扩展**
   - 每个领域可以独立丰富场景
   - 不影响其他领域的分类

4. **对齐商业产品**
   - 钉钉：OA、审批、CRM、HR
   - 飞书：办公、审批、客户、人事
