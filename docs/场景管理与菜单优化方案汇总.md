### 1. 场景管理优化方案

**方案E（推荐）**：

```

│
├── 我的空间 📁       ← 原"文件"
│
```

**关键决策**：

### 3. 菜单结构设计方案（v4）

**核心设计理念**：

```
社区版菜单（5项）：
├─ 首页          ← 综合看板
├─ 办公          ← 办公场景
├─ 我的领域      ← 专业标签 + 推荐
├─ 我的空间      ← 文件 + 资源
└─ 设置          ← 个人管理

企业版菜单（6项）：
├─ 首页
├─ 办公
├─ 我的领域
├─ 我的空间
├─ 知识库        ← 企业版专属
└─ 设置
```

**首页看板设计**：

```
首页看板功能：
├─ 🎯 常用场景      （快速访问）
├─ 📊 数据与分析    （使用统计）
├─ 🔔 消息通知      （未来功能）
├─ 💡 推荐场景      （智能推荐）
└─ 📰 领域新闻      （基于职能标签）
```

**我的领域概念**：
- 用户选择专业标签（人力资源、财务管理、项目管理等）
- 系统个性化推荐相关场景
- 集成领域专属知识库

## 🔑 关键决策记录

### 决策1：删除"我的场景"菜单

**原因**：
- 和"办公"功能重复
- 失去了"我的领域"的原意
- 菜单冗余

**影响**：
- 需要迁移用户数据（已启用的场景）
- 删除相关代码文件

---

### 决策2：添加"首页"菜单

**原因**：
- 作为一站式工作台
- 集成常用功能和信息
- 未来可扩展（消息通知、数据看板等）

**功能**：
- 个人推荐（基于职能标签）
- 最近使用
- 我的收藏
- 热门场景

---

### 决策3：工作场景添加二级菜单

**原因**：
- 126个场景平铺展示，难以查找
- 按业务领域分组，更清晰
- 支持快速筛选和导航

**分类**：
- 9个业务领域作为二级菜单
- 场景按业务领域归类

---

### 决策4：职能标签推荐系统

**原因**：
- 个性化推荐，提升用户体验
- 减少用户找场景的时间
- 基于专业领域精准推荐

**实现**：
- 用户选择职能标签（17个行业）
- 首页推荐相关场景
- 未来集成领域知识库
