📖 使用说明

CoApis 各模块功能详解。

💬 聊天模块

聊天模块是 CoApis 的核心交互界面。

基本操作

操作说明
新建对话点击 + 按钮
切换 Agent顶部下拉菜单选择
选择模型为每个 Agent 选择不同模型
上传附件点击附件图标，支持文件/图片
语音输入点击麦克风图标（需配置语音转写）

快捷指令

指令说明
/查看可用指令列表
/approve审批通过
/deny审批拒绝
/clear清空上下文
/help查看帮助

🎛️ 控制模块

频道管理

路径：控制 → 频道

管理 CoApis 与外部平台的连接：

渠道说明
企业微信接入企业微信机器人
钉钉接入钉钉机器人
Slack接入 Slack 机器人
Telegram接入 Telegram Bot
Webhook自定义集成

配置步骤：添加频道 → 选择平台 → 填入配置 → 测试连接 → 保存

会话管理

路径：控制 → 会话

  • 查看所有活跃会话
  • 查看会话完整记录
  • 暂停、恢复、结束会话

定时任务

路径：控制 → 定时任务

支持 Cron 表达式，实现自动化任务：

# 每天凌晨 2 点
0 2 * * *

# 每周一上午 9 点
0 9 * * 1

# 每 30 分钟
*/30 * * * *

心跳监控

路径：控制 → 心跳

  • 定期检查 Agent 运行状态
  • 异常时发送告警通知
  • 支持异常后自动重启

📁 工作区模块

我的空间

路径：工作区 → 我的空间

个人文件管理区域，每个用户完全隔离。

  • 支持多种文件格式上传/下载
  • 文件夹管理、文件搜索
  • 文件版本历史
  • 同名文件覆盖确认

技能管理

路径：工作区 → 技能

管理 Agent 可用的技能：

内置技能说明
file_reader读取文本文件
pdfPDF 文件处理
xlsxExcel 文件处理
docxWord 文档处理
pptxPowerPoint 处理
browser_use浏览器自动化
web_search网络搜索
markitdown文档转 Markdown
skill-creator技能创建工具

工具管理

管理 Agent 可用的内置工具（56+ 个）：

类别工具
文件操作read_file, write_file, edit_file, grep_search, glob_search
Shell 执行execute_shell_command
浏览器browser_use, desktop_screenshot
多媒体view_image, view_video
记忆memory_search
通信chat_with_agent, submit_to_agent

MCP 客户端

路径：工作区 → MCP

配置 MCP（Model Context Protocol）服务器连接：

{
  "name": "filesystem",
  "transport": "stdio",
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-filesystem", "/data"]
}

ACP 跨 Agent 通信

路径：工作区 → ACP

管理 Agent 间通信：发现其他 Agent、发送消息、传递上下文。

⚙️ 设置模块

智能体管理

路径：设置 → 智能体管理

  • 创建、编辑、删除 Agent
  • 配置 Agent 参数
  • 启用/禁用 Agent

模型配置

路径：设置 → 模型

  • 查看所有可用模型
  • 配置模型参数
  • 测试模型可用性

安全设置

路径：设置 → 安全

  • 审计日志 — 所有操作记录
  • 权限管理 — 用户权限控制
  • IP 白名单 — 访问限制
  • 密码策略 — 密码强度要求
  • 会话超时 — 自动登出时间

Token 消耗

路径：设置 → Token 消耗

统计维度：按用户、按 Agent、按模型、按时间趋势。

备份恢复

路径：设置 → 备份

# 手动备份
docker compose exec server python backup.py --full

# 自动备份（Cron）
0 2 * * * docker compose exec server python backup.py --full

进化系统

路径：设置 → 进化

  • 查看进化历史和知识沉淀
  • 查看技能晋升记录
  • 跨 Agent 知识共享

系统监控

路径：设置 → 系统监控

监控 CPU、内存、磁盘、网络流量、服务状态、请求延迟。
