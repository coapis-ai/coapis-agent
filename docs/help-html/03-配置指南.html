⚙️ 配置指南

三层配置体系：环境变量 → Agent 配置 → Provider 配置。

配置层级

环境变量 (.env)         → 全局配置（LLM 地址、认证、端口等）
Agent 配置 (config.json) → Agent 行为（模型、记忆、工具调用方式）
Provider 配置 (providers.json) → LLM 服务（API 地址、密钥）

优先级：环境变量 > 配置文件 > 内置默认值

目录结构

{WORKING_DIR}/
├── system/              ← 系统级文件（全局共享）
│   ├── config.json      ← Agent 配置
│   ├── providers.json   ← Provider 配置
│   ├── users.json       ← 用户数据
│   ├── auth.json        ← 认证配置
│   ├── permissions.json ← 权限配置
│   └── .secret/         ← 密钥存储
├── workspaces/          ← 用户级数据（按用户隔离）
│   └── {username}/      ← 每个用户独立工作区
├── agents/              ← 全局共享 Agent 模板
├── skills/              ← 全局共享技能模板
└── logs/                ← 运行日志

环境变量配置

配置文件位置：docker/.env

核心配置

变量类型默认值说明
COAPIS_LLM_BASE_URLstring-LLM API 地址（必填）
COAPIS_AUTH_ENABLEDboolTrue启用认证系统
COAPIS_USER_SYSTEM_ENABLEDboolTrue启用多租户用户体系
COAPIS_WORKING_DIRstring/apps/ai/coapis工作目录
COAPIS_PORTint8000后端端口
COAPIS_FRONTEND_PORTint4200前端端口

LLM 限流配置

变量默认值说明
COAPIS_LLM_MAX_CONCURRENT10最大并发 LLM 调用数
COAPIS_LLM_MAX_QPM600每分钟最大查询数
COAPIS_LLM_MAX_RETRIES3最大重试次数

Token 配额（每月）

变量默认值说明
COAPIS_USER_TOKEN_QUOTA_L0100,000L0 用户月配额
COAPIS_USER_TOKEN_QUOTA_L11,000,000L1 用户月配额
COAPIS_USER_TOKEN_QUOTA_L25,000,000L2 用户月配额
COAPIS_USER_TOKEN_QUOTA_L320,000,000L3 用户月配额
COAPIS_USER_TOKEN_QUOTA_L4-1L4 用户（无限制）

限流（每分钟请求数）

变量L0L1L2L3L4
默认值1010502001000

Agent 配置

配置文件位置：{WORKING_DIR}/system/config.json

基本结构

{
  "agents": {
    "active_agent": "default",
    "profiles": {
      "default": {
        "model": "qwen3.6-27b",
        "provider": "local_llm",
        "enable_memory": false,
        "tool_calling_method": "raw"
      }
    }
  }
}

多 Agent 配置

{
  "agents": {
    "active_agent": "default",
    "profiles": {
      "default": {
        "model": "qwen3.6-27b",
        "provider": "local_llm",
        "enable_memory": false,
        "tool_calling_method": "raw"
      },
      "coder": {
        "model": "deepseek-coder-v2",
        "provider": "openai",
        "enable_memory": true,
        "tool_calling_method": "auto"
      }
    }
  }
}

Provider 配置

配置文件位置：{WORKING_DIR}/system/providers.json

本地 LLM

{
  "local_llm": {
    "id": "local_llm",
    "type": "openai",
    "api_base": "http://172.16.6.241:8082/v1",
    "api_key": "none"
  }
}

OpenAI

{
  "openai": {
    "id": "openai",
    "type": "openai",
    "api_base": "https://api.openai.com/v1",
    "api_key": "sk-xxx"
  }
}

阿里云通义千问

{
  "dashscope": {
    "id": "dashscope",
    "type": "openai",
    "api_base": "https://dashscope.aliyuncs.com/compatible-mode/v1",
    "api_key": "sk-xxx"
  }
}
