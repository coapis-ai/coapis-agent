🚀 快速入门

30 秒让 CoApis 跑起来。

一键安装（推荐）

执行安装命令

在你的 Linux 服务器上执行：

curl -fsSL https://raw.githubusercontent.com/coapis/coapis/main/install.sh | bash
💡 前置要求
需要 Linux 系统（Ubuntu 20.04+ / CentOS 8+ / Debian 11+）和 Docker 20.10+。
打开浏览器访问
http://你的服务器IP:4200
登录
字段值
用户名admin
密码admin123
⚠️ 请立即修改默认密码
首次登录后请到 设置 → 个人资料 修改密码。

手动部署

克隆项目
mkdir -p /opt/coapis && cd /opt/coapis
wget -qO- https://raw.githubusercontent.com/coapis/coapis/main/install.sh | bash
配置 LLM 服务

编辑 docker/.env，填写你的 LLM API 地址：

# 使用 Ollama
COAPIS_LLM_BASE_URL=http://localhost:11434/v1

# 使用 OpenAI
COAPIS_LLM_BASE_URL=https://api.openai.com/v1
OPENAI_API_KEY=sk-...
启动服务
cd docker && docker compose up -d
访问

打开 http://你的服务器IP:4200，用 admin / admin123 登录。

首次使用

✅ 登录后你可以
  • 在聊天界面与 AI 对话
  • 上传文件到"我的空间"
  • 在"技能"中查看 AI 已掌握的技能
  • 在"设置"中配置频道、用户、安全等
