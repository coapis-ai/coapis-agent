## 一、场景分类框架

### 1.1 三类场景体系

```
CoApis 场景体系
│
├─ 一类：公用性管理（基础通用）
│   ├─ 办公协作
│   ├─ 行政审批
│   ├─ 组织管理
│   ├─ 人事管理
│   ├─ 党建工作
│   └─ 基础专项
│
├─ 二类：专业领域（行业应用）
│   ├─ 项目管理
│   ├─ 业务领域
│   ├─ 技术领域
│   ├─ 行业应用
│   └─ 专业方案
│
└─ 三类：深度领域（高级专业）
    ├─ 建设实施
    ├─ 评测评估
    ├─ 合法合规
    ├─ 审议审定
    └─ 专业服务
```

### 1.2 场景深度定义

| 类别 | 深度 | 用户群体 | 技术要求 | 实施难度 |
|------|------|---------|---------|---------|
| 一类 | 基础通用 | 全员 | 低 | 易 |
| 二类 | 专业应用 | 专业人员 | 中 | 中 |
| 三类 | 深度专业 | 专家 | 高 | 难 |
