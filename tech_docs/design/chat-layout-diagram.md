## 🤔 问题分析

### 问题1：顶部栏按钮过多

**现状**：
- 左侧4个按钮：🎨 显示设置、🔍 搜索、📋 历史、➕ 新对话
- 右侧2个功能：📋 计划面板、模型选择器

**问题**：
- ⚠️ 按钮过多，视觉拥挤
- ⚠️ 功能分散，不够直观
- ⚠️ 移动端空间有限

---

### 问题2：输入区域功能单一

**现状**：
- 只有 📎 附件按钮
- 缺少文件引用、知识库引用等功能

**问题**：
- ⚠️ 无法引用已有文件
- ⚠️ 无法引用知识库
- ⚠️ 功能受限

---

### 问题3：顶部栏 vs 输入区域分工不明确

**现状**：
- 顶部栏：全局操作（设置、搜索、历史、新对话）
- 输入区域：消息操作（附件、发送）

**问题**：
- ⚠️ 分工明确，但缺少"上下文引用"能力
- ⚠️ 无法在发送消息时引用文件/知识库

### 动态工具配置

```typescript
// 根据场景配置不同工具
function getToolsByScene(scene: 'chat' | 'embedded' | 'card'): ToolbarTool[] {
  const baseTools = [
    { key: 'file-picker', icon: <PaperClipOutlined />, label: '文件', order: 1 },
    { key: 'knowledge-picker', icon: <BookOutlined />, label: '知识库', order: 2 },
  ];

  if (scene === 'chat') {
    // 完整聊天页面：所有工具
    return [
      { key: 'display-settings', icon: <BgColorsOutlined />, label: '设置', order: 0 },
      ...baseTools,
      { key: 'voice', icon: <AudioOutlined />, label: '语音', order: 10 },
    ];
  }

  if (scene === 'embedded') {
    // 嵌入式聊天：简化工具
    return baseTools;
  }

  if (scene === 'card') {
    // 卡片式聊天：最少工具
    return [{ key: 'file-picker', icon: <PaperClipOutlined />, label: '文件' }];
  }

  return baseTools;
}
```

## ✅ 总结

### 调整建议

1. **顶部栏精简**：
   - 保留：历史、新对话、计划面板、模型选择器
   - 移除：显示设置、搜索（移到输入区域工具栏）

2. **输入区域增强**：
   - 新增：折叠式工具栏
   - 包含：显示设置、搜索、文件引用、知识库引用、语音输入等
   - 可配置：插件化，支持动态添加/移除

3. **核心优势**：
   - ✅ 顶部栏更简洁
   - ✅ 功能集中且可扩展
   - ✅ 移动端友好
   - ✅ 默认简洁，按需展开
