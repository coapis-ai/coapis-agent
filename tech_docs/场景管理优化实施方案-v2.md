#### 任务2.2：场景推荐API（0.2天）

**修改文件**：`server/coapis/app/routers/scenes.py`

**API接口**：

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/scenes/recommendations` | 推荐场景（基于职能标签） |
| GET | `/api/scenes/recent` | 最近使用场景 |
| GET | `/api/scenes/hot` | 热门场景 |
| POST | `/api/scenes/favorite/{scene_id}` | 收藏场景 |
| DELETE | `/api/scenes/favorite/{scene_id}` | 取消收藏 |
| GET | `/api/scenes/favorites` | 收藏列表 |

**推荐算法**：
```python
def get_recommendations(function_tags: List[str], limit: int = 6):
    """
    基于职能标签推荐场景
    
    算法：
    1. 获取用户职能标签
    2. 匹配场景的tag_ids
    3. 按匹配度排序
    4. 返回前N个场景
    """
    scenes = load_scenes()
    
    # 计算匹配度
    scored_scenes = []
    for scene in scenes:
        score = len(set(function_tags) & set(scene.tag_ids))
        if score > 0:
            scored_scenes.append((scene, score))
    
    # 排序并返回
    scored_scenes.sort(key=lambda x: x[1], reverse=True)
    return [s[0] for s in scored_scenes[:limit]]
```

#### 任务3.2：实现数据持久化（0.2天）

**本地存储同步**：
```typescript
// 用户配置同步
const syncUserPreferences = async () => {
  // 1. 从本地存储读取
  const localPrefs = localStorage.getItem(STORAGE_KEYS.USER_PREFERENCES);
  
  // 2. 同步到服务器
  await api.post('/api/user/preferences', localPrefs);
  
  // 3. 从服务器获取最新
  const serverPrefs = await api.get('/api/user/preferences');
  
  // 4. 更新本地存储
  localStorage.setItem(STORAGE_KEYS.USER_PREFERENCES, JSON.stringify(serverPrefs));
};
```

**关键数据**：
- 职能标签（function_tags）
- 最近使用（recent_scenes）
- 收藏场景（favorite_scenes）
