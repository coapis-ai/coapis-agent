## 💾 数据库设计

### 用户配置表

```sql
CREATE TABLE user_preferences (
    id SERIAL PRIMARY KEY,
    user_id VARCHAR(100) NOT NULL UNIQUE,
    function_tags TEXT[],
    recent_scenes JSONB,
    favorite_scenes TEXT[],
    dashboard_config JSONB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_user_preferences_user_id ON user_preferences(user_id);
```

### 场景使用统计表

```sql
CREATE TABLE scene_usage_stats (
    id SERIAL PRIMARY KEY,
    scene_id VARCHAR(100) NOT NULL,
    user_id VARCHAR(100) NOT NULL,
    usage_count INTEGER DEFAULT 0,
    last_used_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(scene_id, user_id)
);

CREATE INDEX idx_scene_usage_stats_scene_id ON scene_usage_stats(scene_id);
CREATE INDEX idx_scene_usage_stats_user_id ON scene_usage_stats(user_id);
```

## 🚀 部署计划

### 开发环境

```bash
# 1. 构建前端
cd client && npm run build

# 2. 复制到容器
docker cp client/dist/. coapis-server-dev:/app/coapis/console/

# 3. 重启服务
docker restart coapis-server-dev
```

### 生产环境

```bash
# 1. 构建镜像
docker build -t coapis-server:v0.11.0 .

# 2. 更新配置
# 修改 docker-compose.yml 中的镜像版本

# 3. 重启服务
docker compose -f docker-compose.yml up -d
```
