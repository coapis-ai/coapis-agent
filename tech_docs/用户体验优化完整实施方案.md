## 一、项目概述

### 1.1 项目背景

CoApis 是一个企业级多用户多智能体平台，当前用户体验评分为 **2.6/5**，存在以下核心问题：

| 问题类别 | 具体问题 | 影响程度 | 优先级 |
|---------|---------|---------|--------|
| **权限混乱** | 普通用户可以删除模型配置 | 🔥🔥🔥 高 | P0 |
| **入口分散** | 工作台和控制台平级 | 🔥🔥🔥 高 | P0 |
| **菜单过多** | 22个菜单项，视觉负担重 | 🔥🔥 中 | P1 |
| **命名抽象** | MCP、进化等专业术语 | 🔥🔥 中 | P1 |
| **缺少概览** | 管理员无法快速了解系统状态 | 🔥🔥 中 | P1 |

### 1.2 项目目标

**核心目标**：实现双入口模式，分离用户工作入口和管理入口

**具体目标**：
1. ✅ 普通用户首次使用时长从5分钟降低到1分钟
2. ✅ 管理员配置效率提升50%
3. ✅ 权限错误率从5%降低到0.1%
4. ✅ 用户满意度从70%提升到90%

### 1.3 项目范围

**包含范围**：
- 权限系统优化
- 菜单结构重组
- 工作台重新设计
- 控制台重新设计
- 移动端优化

**不包含范围**：
- 后端架构重构
- 数据库迁移
- 第三方集成

```typescript
// 工作台布局组件
function WorkbenchLayout({ children }) {
  const { user } = useUser();
  
  return (
    <Layout className="workbench-layout">
      <WorkbenchHeader />
      <Layout>
        <WorkbenchSidebar />  {/* 5个菜单项 */}
        <Content>{children}</Content>
      </Layout>
      <EntrySwitchButton />  {/* 切换到控制台 */}
    </Layout>
  );
}

// 控制台布局组件
function ConsoleLayout({ children }) {
  const { user } = useUser();
  
  // 普通用户只读访问控制台
  if (user.role === 'user') {
    return <ReadOnlyConsole>{children}</ReadOnlyConsole>;
  }
  
  return (
    <Layout className="console-layout">
      <ConsoleHeader />
      <Layout>
        <ConsoleSidebar />  {/* 15个菜单项 */}
        <Content>{children}</Content>
      </Layout>
      <EntrySwitchButton />  {/* 切换到工作台 */}
    </Layout>
  );
}

// 入口切换按钮
function EntrySwitchButton() {
  const location = useLocation();
  const navigate = useNavigate();
  
  const isInWorkbench = location.pathname.startsWith('/workbench');
  const targetPath = isInWorkbench ? '/console' : '/workbench';
  const targetLabel = isInWorkbench ? '控制台' : '工作台';
  
  return (
    <Button
      className="entry-switch-button"
      onClick={() => navigate(targetPath)}
    >
      切换到{targetLabel}
    </Button>
  );
}

// 权限守卫组件
function PermissionGuard({ 
  module, 
  action, 
  children,
  fallback = null 
}) {
  const { hasPermission } = usePermission();
  
  if (!hasPermission(module, action)) {
    return fallback;
  }
  
  return children;
}

// 使用示例
<PermissionGuard module="models" action="update" fallback={<ReadOnlyView />}>
  <ModelEditor />
</PermissionGuard>
```

```python
        "recent_scenes": scene_service.get_recent_scenes(user.id),

    }

async def start_scene_chat(
    scene_id: str,
    user: User = Depends(get_current_user)
):
    """启动场景对话"""
    return chat_service.start_scene_chat(scene_id, user.id)

# 控制台API（管理员权限）
@router.get("/console/dashboard")
@require_permission("dashboard", "read")
async def get_dashboard(user: User = Depends(get_current_user)):
    """获取控制台概览"""
    return dashboard_service.get_stats()

@router.post("/console/scenes")

    scene_data: SceneCreate,
```
