## 🔧 技术要点

### 1. 路由顺序修复

**问题**：FastAPI按定义顺序匹配路由，`/recommendations`被`/{scene_id}`匹配

**解决**：调整路由顺序
```python
# ✅ 正确顺序
@router.get("/recommendations")  # 固定路径在前
@router.get("/hot")              # 固定路径在前
@router.get("/{scene_id}")       # 参数化路由在后
```

---

### 2. 认证Token处理

```typescript
// 从localStorage获取token
const token = localStorage.getItem('coapis_auth_token');

// API调用携带token
const response = await fetch('/api/user/scene-preferences/function-tags', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});
```

---

### 3. 路径参数获取

```typescript
// 使用useParams获取路径参数
import { useParams } from 'react-router-dom';

const { category } = useParams<{ category?: string }>();
const categoryParam = category || 'all';
```
