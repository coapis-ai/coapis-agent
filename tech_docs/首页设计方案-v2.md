## 📋 设计原则

### 核心理念

1. **简洁至上**：首页是一个快捷入口，不是功能中心
2. **突出重点**：推荐场景、最近使用、我的收藏是核心
3. **视觉清晰**：卡片式布局，层次分明
4. **交互友好**：一键直达，减少点击次数

```typescript
// 基于用户职能标签推荐场景
const getRecommendedScenes = async () => {
  const userTags = await getUserFunctionTags();
  
  // 1. 筛选匹配职能标签的场景
  const matchedScenes = allScenes.filter(scene => 
    scene.tags.some(tag => userTags.includes(tag))
  );
  
  // 2. 按使用频率排序
  const sortedScenes = matchedScenes.sort(
    (a, b) => b.usageCount - a.usageCount
  );
  
  // 3. 返回前8个
  return sortedScenes.slice(0, 8);
};
```

### 3. 最近使用

**数据来源**：
- 从用户配置中读取 `recentScenes` 字段
- 每次启动场景时，更新时间戳

**数据结构**：
```typescript
interface RecentScene {
  sceneId: string;
  lastUsedAt: string; // ISO时间戳
  usageCount: number;
}

// 用户配置
interface UserPreferences {
  functionTags: string[];      // 职能标签
  recentScenes: RecentScene[]; // 最近使用
  favoriteScenes: string[];    // 收藏场景
}
```

**排序逻辑**：
```typescript
// 按最近使用时间排序
const sortedRecentScenes = userPreferences.recentScenes
  .sort((a, b) => new Date(b.lastUsedAt) - new Date(a.lastUsedAt))
  .slice(0, 8);
```

### 4. 我的收藏

**收藏功能**：
```typescript
// 添加收藏
const handleAddFavorite = async (sceneId: string) => {
  const favorites = [...userPreferences.favoriteScenes, sceneId];
  await updateUserPreferences({ favoriteScenes: favorites });
  message.success('已添加到收藏');
};

// 取消收藏
const handleRemoveFavorite = async (sceneId: string) => {
  const favorites = userPreferences.favoriteScenes.filter(id => id !== sceneId);
  await updateUserPreferences({ favoriteScenes: favorites });
  message.success('已取消收藏');
};

// 判断是否已收藏
const isFavorite = (sceneId: string) => {
  return userPreferences.favoriteScenes.includes(sceneId);
};
```
