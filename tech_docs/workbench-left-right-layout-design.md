# 工作台左右分栏设计

> **核心设计**：左侧菜单栏 + 右侧内容窗口  
> **优势**：结构清晰、易于扩展、符合经典交互模式  
> **创建日期**：2026-07-17

### 1.2 为什么选择左右分栏？

**经典交互模式**：
- ✅ 邮件应用：左侧文件夹列表 + 右侧邮件列表
- ✅ 文件管理器：左侧目录树 + 右侧文件列表
- ✅ 云存储应用：左侧分类 + 右侧文件列表
- ✅ 企业应用：左侧功能菜单 + 右侧工作区

**优势**：
- ✅ 结构清晰，左侧导航，右侧内容
- ✅ 易于扩展，左侧菜单可以放更多内容
- ✅ 符合用户习惯，不需要学习成本
- ✅ 响应式设计友好（移动端可折叠菜单）

## 二、左侧菜单栏设计

### 2.1 菜单结构

```
左侧菜单栏
│
├── 业务场景（分组标题）
│   ├── 📁 办公
│   ├── 📊 数据分析
│   ├── 📄 文档处理
│   └── 💬 沟通协作
│
├── ──────────（分隔线）
│
└── 快速访问（分组标题）
    ├── ⭐ 我的收藏
    ├── 📝 最近使用
    └── 🔥 推荐场景
```

**说明**：
- **业务场景**：按功能标签分类的场景
- **我的收藏**：用户手动收藏的场景
- **最近使用**：用户最近使用的场景
- **推荐场景**：系统智能推荐的场景

---

### 2.2 菜单项设计

```
┌────────────┐
│ 📁 办公    │ ← 图标 + 名称
│   12个场景 │ ← 副标题（场景数量，可选）
└────────────┘
```

**交互**：
- 点击菜单项 → 右侧内容区切换显示对应场景
- 当前选中项高亮
- hover 显示背景色

**状态**：
- 正常：默认颜色
- hover：背景色高亮
- 选中：背景色 + 左侧边框高亮

## 五、交互设计

### 5.1 菜单切换

**流程**：

```
用户点击左侧菜单"数据分析"
    ↓
菜单项高亮
    ↓
右侧内容区切换显示"数据分析"分区的场景
    ↓
场景卡片从左侧滑入（动画）
```

---

### 5.2 场景卡片交互

**点击"开始使用"**：

```
用户点击场景卡片的"开始使用"按钮
    ↓
场景代入
    ├── 创建聊天会话
    ├── 注入系统提示词
    ├── 加载关联技能
    └── 显示欢迎消息
    ↓
跳转到聊天界面
```

---

### 5.3 收藏场景

**点击收藏按钮（卡片右上角）**：

```
用户点击场景卡片的收藏按钮 ⭐
    ↓
场景添加到"我的收藏"
    ↓
左侧菜单"我的收藏"数量 +1
    ↓
收藏按钮变为实心星星
```

## 八、总结

### 左右分栏设计的优势

| 维度 | 优势 |
|------|------|
| **结构清晰** | 左侧导航，右侧内容，经典布局 |
| **易于扩展** | 左侧菜单可以放更多内容（收藏、最近、推荐等） |
| **符合习惯** | 类似邮件、文件管理器等应用 |
| **响应式友好** | 移动端可折叠菜单 |
| **交互简单** | 点击菜单项切换内容，点击场景卡片代入 |

### 核心设计要点

1. **左侧菜单**：业务场景分类 + 快速访问入口
2. **右侧内容**：场景卡片网格展示
3. **菜单切换**：点击菜单项切换右侧内容
4. **场景卡片**：图标 + 名称 + 描述 + 按钮
