# 首页设计方案（卡片式布局）

**创建时间**：2026-07-23  
**版本**：v3.0（卡片式布局）  
**设计原则**：灵活、可扩展、卡片化

---

## 📋 核心理念

### Widget-based 设计

**可插拔卡片系统**：
- 每个功能模块都是独立的卡片（Widget）
- 用户可以自定义显示哪些卡片
- 卡片可以拖拽排序
- 支持后续扩展（图表、数据看板等）

**设计优势**：
- ✅ 灵活：用户可以自定义首页布局
- ✅ 可扩展：新增功能只需添加新卡片
- ✅ 个性化：不同用户可以有不同的首页
- ✅ 易维护：每个卡片独立开发和测试

## 📐 卡片布局系统

### 卡片类型定义

```typescript
// 卡片类型枚举
enum WidgetType {
  WEATHER = 'weather',           // 天气卡片
  WELCOME = 'welcome',           // 欢迎卡片
  MESSAGES = 'messages',         // 消息通知卡片
  RECOMMENDATIONS = 'recommendations', // 推荐场景卡片
  RECENT = 'recent',             // 最近使用卡片
  FAVORITES = 'favorites',       // 收藏卡片
  HOT = 'hot',                   // 热门场景卡片
  STATISTICS = 'statistics',     // 统计数据卡片
  CHARTS = 'charts',             // 图表卡片
  QUICK_ACTIONS = 'quick_actions', // 快捷操作卡片
  ANNOUNCEMENTS = 'announcements', // 公告卡片
  CUSTOM = 'custom',             // 自定义卡片
}

// 卡片配置接口
interface WidgetConfig {
  id: string;
  type: WidgetType;
  title: string;
  icon: string;
  size: 'small' | 'medium' | 'large'; // 卡片尺寸
  position: { x: number; y: number }; // 网格位置
  visible: boolean;                    // 是否显示
  collapsible: boolean;                // 是否可折叠
  refreshInterval?: number;            // 自动刷新间隔（秒）
}
```

### 布局网格系统

**网格配置**：
```
桌面端：12列网格
├─ Small卡片：占3列（25%宽度）
├─ Medium卡片：占6列（50%宽度）
```

### 2. 消息通知卡片

**尺寸**：Large（12列）

**功能**：
- 显示最新消息通知（最多5条）
- 消息类型：系统通知、任务通知、智能体通知
- 点击查看详情
- 支持标记已读

**数据结构**：
```typescript
interface Message {
  id: string;
  type: 'system' | 'task' | 'agent' | 'announcement';
  title: string;
  content: string;
  timestamp: string;
  read: boolean;
  action?: {
    text: string;
    url: string;
  };
}
```

**UI设计**：
```typescript
<div className={styles.messagesCard}>
  <div className={styles.header}>
    <span className={styles.icon}>🔔</span>
    <span className={styles.title}>最新消息</span>
    <Button type="link" size="small" onClick={handleViewAll}>
      查看全部 →
    </Button>
  </div>
  
  <div className={styles.content}>
    {messages.map(msg => (
      <div 
        key={msg.id} 
        className={styles.messageItem}
        onClick={() => handleMessageClick(msg.id)}
      >
        <span className={styles.icon}>
          {getMessageIcon(msg.type)}
        </span>
        <div className={styles.text}>
          <div className={styles.title}>{msg.title}</div>
          <div className={styles.time}>{formatTime(msg.timestamp)}</div>
        </div>
        {!msg.read && <span className={styles.unread} />}
      </div>
    ))}
    
    {messages.length === 0 && (
      <div className={styles.empty}>
        暂无新消息
      </div>
    )}
  </div>
</div>
```

**消息类型图标**：
```typescript
const getMessageIcon = (type: string) => {
  const icons = {
    'system': '📢',      // 系统通知
    'task': '⏰',        // 任务通知
    'agent': '🤖',       // 智能体通知
    'announcement': '📰', // 公告
  };
  return icons[type] || '📌';
};
```

### 卡片拖拽排序

**使用库**：react-dnd 或 dnd-kit

```typescript
import { DndContext, closestCenter } from '@dnd-kit/core';
import { arrayMove, SortableContext, rectSortingStrategy } from '@dnd-kit/sortable';

function Dashboard() {
  const [widgets, setWidgets] = useState(initialWidgets);
  
  const handleDragEnd = (event) => {
    const { active, over } = event;
    
    if (active.id !== over.id) {
      setWidgets((widgets) => {
        const oldIndex = widgets.findIndex(w => w.id === active.id);
        const newIndex = widgets.findIndex(w => w.id === over.id);
        
        return arrayMove(widgets, oldIndex, newIndex);
      });
    }
  };
  
  return (
    <DndContext collisionDetection={closestCenter} onDragEnd={handleDragEnd}>
      <SortableContext items={widgets} strategy={rectSortingStrategy}>
        <div className={styles.dashboard}>
          {widgets.map(widget => (
            <SortableWidget key={widget.id} widget={widget} />
          ))}
        </div>
      </SortableContext>
    </DndContext>
  );
}
```
