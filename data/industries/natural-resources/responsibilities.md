# 自然资源和规划局职责梳理

> **领域**：自然资源和规划
> **更新日期**：2026-07-17
> **信息来源**：三定方案、法律法规、政府网站

---

## 一、主要职责（法定职责）

### 1.1 自然资源资产管理

**职责描述**：
- 履行全民所有土地、矿产、森林、草原、湿地、水等自然资源资产所有者职责
- 负责自然资源资产价值评估和核算
- 建立自然资源资产台账和报告制度

**具体工作**：
- 自然资源资产清查统计
- 自然资源资产价值评估
- 自然资源资产收益管理
- 自然资源资产报告编制

---

### 1.2 国土空间用途管制

**职责描述**：
- 履行所有国土空间用途管制职责
- 制定国土空间用途管制制度和技术标准
- 监督落实国土空间用途管制要求

**具体工作**：
- 国土空间用途转用审批
- 建设用地计划管理
- 耕地保护监管
- 生态保护红线管理

---

### 1.3 自然资源调查监测

**职责描述**：
- 负责自然资源调查监测评价
- 建立统一自然资源调查监测评价制度
- 组织实施自然资源基础调查、专项调查和监测

**具体工作**：
- 国土调查（年度变更调查、专项调查）
- 自然资源监测（遥感监测、实地核查）
- 调查成果整理和分析
- 调查数据库建设和维护

---

### 1.4 自然资源确权登记

**职责描述**：
- 负责自然资源统一确权登记工作
- 指导监督自然资源和不动产确权登记
- 建立自然资源和不动产登记信息管理平台

**具体工作**：
- 不动产登记（首次登记、变更登记、转移登记、注销登记）
- 自然资源确权登记
- 权属调查和纠纷调处
- 登记信息查询和共享

---

### 1.5 自然资源有偿使用

**职责描述**：
- 负责自然资源资产有偿使用工作
- 建立自然资源市场交易规则
- 监督管理自然资源市场

**具体工作**：
- 土地使用权出让、租赁、作价出资
- 矿业权出让、转让
- 自然资源价格评估
- 自然资源市场监测监管

---

### 1.6 国土空间规划

## 二、业务模块职责详解

### 2.1 国土空间规划

#### 2.1.1 规划编制

**主要内容**：
- 国土空间总体规划（市、县、乡镇）
- 控制性详细规划
- 专项规划（交通、市政、生态等）
- 村庄规划

**工作流程**：
```
规划启动 → 资料收集 → 现状调研 → 方案编制
    ↓
专家评审 → 公示征求意见 → 修改完善
    ↓
报批审查 → 批准实施 → 备案归档
```

**关键文档**：
- 规划编制任务书
- 规划现状调研报告
- 规划方案（文本、图纸、说明书）
- 专家评审意见
- 公众意见处理报告
- 规划批准文件

---

#### 2.1.2 规划审查

**主要内容**：
- 建设项目规划选址
- 建设用地规划许可
- 建设工程规划许可
- 乡村建设规划许可

**审查要点**：
- 符合国土空间规划
- 符合规划技术标准
- 符合生态保护要求
- 符合耕地保护要求

**关键文档**：
- 规划选址论证报告
- 规划条件通知书
- 建设用地规划许可证
- 建设工程规划许可证

---

#### 2.1.3 规划调整

**主要内容**：
- 规划修改（重大调整）
- 规划修编（局部调整）
- 规划动态维护

**调整条件**：
- 上位规划调整
- 重大项目建设需要
- 公共利益需要
- 规划实施评估结果

**工作流程**：
```
调整申请 → 必要性论证 → 方案编制
    ↓
专家评审 → 公示征求意见 → 批准实施
```

---

### 2.2 土地利用管理

#### 2.2.1 用地审批

**主要内容**：
- 建设项目用地预审
- 农用地转用审批
- 土地征收审批
- 供地审批

**审批流程**：
```
项目立项 → 用地预审 → 规划选址
    ↓
农用地转用 → 土地征收 → 供地审批
    ↓
登记发证 → 跟踪监管
```

**关键材料**：
- 建设项目用地预审申请表
- 项目建议书或可行性研究报告
- 规划选址意见书
- 土地利用现状图
- 勘测定界报告
- 征地补偿安置方案
- 供地申请文件

---

#### 2.2.2 土地供应

**供应方式**：
- 划拨（公益项目）
- 出让（经营性项目）
- 租赁（短期用地）
- 作价出资（国企改革）

**出让流程**：
```
地块策划 → 地价评估 → 出让方案
    ↓
批准实施 → 公告招拍挂 → 确认成交
    ↓
签订合同 → 缴纳出让金 → 登记发证
```

**关键文档**：
- 土地出让方案
- 土地评估报告
- 招拍挂公告
- 成交确认书
- 国有建设用地使用权出让合同

---

#### 2.2.3 耕地保护

**保护措施**：
- 耕地保护责任目标考核
- 基本农田划定和保护
- 耕地占补平衡
- 耕地进出平衡
- 耕地质量等别更新

**占补平衡流程**：
```
占用耕地 → 补充耕地指标落实 → 补充耕地项目验收
    ↓
指标核销 → 登记备案
```

**关键文档**：
- 耕地保护责任书
- 基本农田保护责任卡
- 补充耕地项目验收文件
- 耕地占补平衡台账

---

### 2.3 自然资源调查监测

#### 2.3.1 国土调查

**调查类型**：
- 全国国土调查（十年一次）
- 年度变更调查（每年）
- 专项调查（按需）

**调查内容**：
- 地类调查（耕地、林地、草地、建设用地等）
- 权属调查（国有、集体）
- 坡度调查
- 田坎系数调查

**工作流程**：
```
资料准备 → 影像获取 → 内业判读
    ↓
外业调查 → 数据建库 → 质量检查
    ↓
成果汇总 → 上报审查 → 数据库更新
```

---

#### 2.3.2 自然资源监测

**监测方式**：
- 遥感监测（卫星遥感、无人机航拍）
- 实地核查（定期巡查、专项检查）
- 变更调查（年度更新）

**监测重点**：
- 耕地保护监测
- 违法用地监测
- 生态保护红线监测
- 国土空间规划实施监测

**监测成果**：
- 监测图斑数据
- 变化信息提取
- 监测报告
- 统计分析表

---

### 2.4 不动产登记

#### 2.4.1 登记类型

**主要类型**：
- 首次登记

**调查成果**：

## 三、典型工作场景

### 3.1 规划编制场景

**场景**：编制某县国土空间总体规划

**工作流程**：
1. 成立规划编制领导小组和工作专班
2. 收集基础资料（现状数据、规划数据、专题数据）
3. 开展现状调研和专题研究
4. 编制规划方案（初稿、征求意见稿、送审稿、报批稿）
5. 组织专家评审和技术审查
6. 公示征求意见
7. 报批审查和备案

**涉及科室**：
- 国土空间规划科（牵头）
- 办公室（后勤保障）
- 政策法规科（合法性审查）
- 其他相关科室（资料提供、专项协调）

**关键文档**：
- 规划编制任务书
- 现状调研报告
- 专题研究报告
- 规划文本、图纸、说明书
- 公众意见处理报告
- 专家评审意见
- 规划批准文件

---

### 3.2 用地审批场景

**场景**：某重点建设项目用地审批

**工作流程**：
1. 项目立项和选址论证
2. 用地预审
3. 农用地转用和土地征收审批
4. 供地审批
5. 登记发证

**涉及科室**：
- 国土空间用途管制科（牵头）
- 国土空间规划科（规划审查）
- 自然资源调查监测科（现状核查）
- 自然资源确权登记科（登记发证）

**关键文档**：
- 项目建议书或可行性研究报告
- 规划选址论证报告
- 用地预审意见
- 农用地转用方案
- 土地征收方案
- 勘测定界报告
- 供地审批文件
- 不动产权证书

---

### 3.3 执法监察场景

**场景**：查处违法占地建设案件

**工作流程**：
1. 巡查发现或群众举报
2. 现场制止和调查取证
3. 立案查处
4. 作出行政处罚决定
5. 申请法院强制执行（如需要）
6. 结案归档

**涉及科室**：
- 执法监察科（牵头）
- 国土空间规划科（规划认定）
- 国土空间用途管制科（用地认定）
- 政策法规科（法制审查）

**关键文档**：
- 现场检查笔录
- 询问笔录
- 违法现场照片和测绘报告
- 案件调查报告
- 行政处罚告知书
- 行政处罚决定书
- 强制执行申请书
- 结案报告

## 四、专业术语

### 4.1 规划类术语

| 术语 | 含义 |
|------|------|
| **国土空间规划** | 对一定区域国土空间开发保护作出的空间布局安排 |
| **"三区三线"** | 城镇空间、农业空间、生态空间；城镇开发边界、永久基本农田保护红线、生态保护红线 |
| **规划条件** | 对建设项目提出的规划建设要求 |
| **控制性详细规划** | 对具体地块的建设控制要求 |
| **规划修改** | 对已批准规划的重大调整 |
| **规划动态维护** | 对规划数据的日常更新 |

### 4.2 土地类术语

| 术语 | 含义 |
|------|------|
| **农用地转用** | 将农用地转为建设用地 |
| **土地征收** | 将集体土地征收为国有土地 |
| **占补平衡** | 占用耕地与补充耕地数量质量相当 |
| **进出平衡** | 耕地转为其他农用地与整治恢复耕地的平衡 |
| **土地供应** | 将国有土地使用权划拨或出让给使用者 |
| **基准地价** | 各用途土地级别或均质区域的平均价格 |

### 4.3 登记类术语

| 术语 | 含义 |
|------|------|
| **不动产** | 土地、海域以及房屋、林木等定着物 |
| **首次登记** | 不动产权利第一次登记 |
| **转移登记** | 因买卖、继承等导致权属转移的登记 |
| **更正登记** | 更正登记簿记载事项错误的登记 |
| **异议登记** | 利害关系人对登记事项提出异议的登记 |
| **预告登记** | 为保障将来实现物权而进行的登记 |

### 4.4 矿产类术语

| 术语 | 含义 |
|------|------|
| **矿业权** | 探矿权和采矿权的合称 |
| **探矿权** | 在勘查许可证规定的范围内勘查矿产资源的权利 |
| **采矿权** | 在采矿许可证规定的范围内开采矿产资源的权利 |
| **矿产资源储量** | 矿产资源在地下的埋藏量 |
| **矿床** | 具有经济价值的矿物集合体 |
| **回采率** | 实际采出的矿石量与矿石储量的比值 |
